PlayStation 2 Guides

FreeMcBoot (FMCB) Installation

FreeMcBoot is the essential softmod for launching homebrew on PS2 fat models and most slims (not SCPH-900xx series).

  1. Download the latest FMCB installer from the official source.
  2. Format a USB drive to FAT32.
  3. Copy the installer files to the root of the USB drive.
  4. You need an entry point: Use FreeDVDBoot (burn a compatible DVD), a compatible game exploit, or an already modded console.
  5. Run the installer and select "Multi-Install" for best cross-console compatibility.
  6. Once installed, insert the memory card – you'll see the FMCB boot screen.
Always use official files. Multi-install works on most models except the latest slims.

Open PS2 Loader (OPL) Setup

OPL lets you load PS2 backups from USB/HDD/SMB and PS1 games via Popstarter.

  1. Download the latest OPL from GitHub.
  2. Place OPL.ELF on your FMCB memory card or storage device.
  3. For PS2 games: Place ISOs in a "DVD" folder (for DVD-based) or "CD" (for CD-based) on your device.
  4. Launch OPL, go to Settings > Enable the device (USB/HDD/SMB).
  5. For PS1 games (Popstarter): Convert games to VCD format, rename ELFs appropriately, place in a POPS folder.
  6. Configure individual game settings in OPL for compatibility modes.

Check the official OPL compatibility list for best settings.

MechaPwn – Region Unlock for PS2 Slim

Makes slim PS2 models (SCPH-500xx to 900xx) region-free for original discs.

  1. Requires FMCB or similar to launch homebrew.
  2. Download MechaPwn ELF from the official GitHub.
  3. Copy to USB and launch via uLaunchELF.
  4. Backup your current MechaCon settings first!
  5. Select the DEX option for full region-free (recommended for most users).
  6. Power cycle and verify.
Reversible but risky – follow instructions carefully. Not all models get full PS1 region-free.
Download MechaPwn

PlayStation 1 Softmods

Popular no-hardware-mod methods to play backups/imports on original PS1 hardware:

Tonyhax International (Recommended)

Best overall compatibility, including anti-piracy bypasses. Uses FreePSXBoot exploit.

  1. Download from official GitHub (alex-free/tonyhax-international).
  2. Prepare a PS1 memory card (use a modded PS2 with FMCB or PC tool like MemcardRex).
  3. Flash the Tonyhax loader via FreePSXBoot method.
  4. Insert card, boot console (lid open if needed), access memory card manager to trigger.
  5. Play burned discs or imports directly.

Superior for games with LibCrypt/SecuROM protection.

Download Tonyhax

FreePSXBoot + UniROM

Simple exploit with built-in tools (e.g., format changer).

  1. Flash a memory card with FreePSXBoot (bundled with UniROM often).
  2. Boot with lid open, go to memory card screen.
  3. UniROM loads – insert disc to play backups.

Good for basic use, but Tonyhax handles more protected games better.

Download FreePSXBoot
Always use legitimate backups. Compatibility varies by PS1 model – check lists on GitHub repos.

Hardware Mods and Modchips (PS1 & PS2)

Hardware mods require soldering and technical skills. These can void warranties and risk damaging your console. Use for preservation purposes only.

PS1 Modchips

Classic chips to bypass protection for backups and imports. Installation involves soldering wires to the motherboard.

  • MM3 (MultiMode 3): Supports backups and region unlock, works on all PS1 models (may have issues on early PU-7/PU-8). Requires 7-8 wires. Installation Guide
  • Mayumi v4: Similar to MM3, reliable on most models except SCPH-100x. Internal clock for better timing.
  • ONEchip: For PAL PSone (SCPH-102), supports BIOS patching.
  • PsNee: Open-source (Atmel-based), stealth mode, works on early models like PU-7/8.
Modchips may be illegal in some regions for circumvention. Check local laws.

PS1 Raspberry Pi Pico Modchip

A modern, cheap ($5) open-source modchip using Raspberry Pi Pico for backups on SCPH-100x models.

  1. Download firmware from community sources (e.g., GitHub repos for Pico mods).
  2. Flash the Pico with the modchip code.
  3. Solder wires to PS1 board (requires skills).
  4. Test for compatibility.
Search GitHub for Firmware
Requires soldering; not for beginners.

PS1 xStation (ODE)

Optical Drive Emulator replaces the CD drive for loading games from SD card. No disc needed.

  1. Purchase xStation kit from reputable sellers.
  2. Install by removing old drive and soldering the board.
  3. Format SD card with games in BIN/CUE format.
xStation GitHub

PS2 Modchips

Hardware chips for direct boot of backups without softmods.

  • ModBo: Open-source, supports all revisions. Discussion on PSX-Place for DIY.
  • Matrix Infinity: Classic chip, region unlock, macro support. Hard to find new.
  • Open-Source Modchip Projects: Community efforts for new chips supporting all motherboards.
PSX-Place Discussion

PS2 SATA Mod

Upgrade fat PS2 network adapter to SATA for faster HDD loading.

  1. Purchase SATA adapter board.
  2. Replace IDE connector in network adapter.
  3. Install SATA HDD and use OPL/FHDB.
Soldering required; compatible with fat models only.

PS2 HDMI Mods

Solder HDMI output for modern TVs (e.g., PS2Digital or similar kits).

  1. Buy HDMI mod kit.
  2. Solder to video chip.
  3. Test output.
PS2 Mods Wiki

MemCard PRO2 (PS1/PS2)

Hardware memory card with infinite virtual saves via SD card.

  1. Purchase from 8BitMods or similar.
  2. Insert SD card with saves.
  3. Use in console slot.
